# Online Retail ML Project

Прогноз покупок и сегментация клиентов интернет‑магазина на основе транзакционных данных.

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/GirenokNikita/Online_retail/blob/main/Online_retail_dataset_clean_final.ipynb)

## Ключевые результаты

- **Сегментация RFM/ABC:** топ‑сегмент — 16,33 % клиентов; клиенты с выкупом < 50 % — низкоприоритетные.
- **Логистическая регрессия:** ROC‑AUC ≈ 0,977, F1‑score = 0,885 — отлично подходит для приоритизации клиентов.
- **Линейная регрессия:** R² = 0,38 (розница), R² = 0,41 (опт) — приоритет задачи в ранжировании, а не в абсолютной точности.

## Что внутри

- Очистка данных: разделение покупок и возвратов, обработка выбросов (IQR), нормализация.
- RFM‑ и ABC‑сегментация, расчёт доли выкупа.
- Обучение и оценка моделей, объединение результатов в итоговую таблицу приоритетов.

## Структура файлов и папок

- `Online_retail.csv.gz` — исходные данные.
- `Online_retail_dataset_clean_final.ipynb` — полный пайплайн анализа и ML (с кодом, проверками и комментариями).
- `ML/` — папка с сохранёнными обученными моделями (для быстрого применения без повторного обучения).
- `Report/` — чистый Jupyter‑ноутбук без кода: только выводы, интерпретации и итоговые рекомендации (удобно для презентации).
  - В папке `Report/` также лежат все визуализации (графики, boxplot, карты сегментов) — их можно сразу вставлять в отчёты и презентации.

## Запуск

1. Нажми кнопку **Open In Colab** выше.
2. В интерфейсе Colab нажми **«Подключиться»** (Connect) в правом верхнем углу.
3. В меню выбери **Runtime → Run All** (или нажми `Ctrl+F9`).

Проект автоматически:
- проверит наличие файла,
- при необходимости скачает данные из репозитория,
- пропустит проблемные строки,
- построит модели,
- сформирует итоговые таблицы.
